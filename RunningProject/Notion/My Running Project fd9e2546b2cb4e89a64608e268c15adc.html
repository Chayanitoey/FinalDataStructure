My Running Project

About this project

Since the beginning of the pandemic, I started to go running twice a week or more to articulate my thoughts about life while maintaining my mental sanity also to help myself going through the unprecedented global crisis...Running has basically been my ‘Therapy’ since then, plus it is free and widely accessible. Every time I go running, I always keep track of my performance. It is really just for fun and it is also insightful for self improvements...Thanks to STRAVA ( I’m a fan of this app) I was able to extract my metadata of what I have run in the past year and analyzed the data into a mega analysis project for Data Structure Course at Parsons (see more details here )

Data

This project wouldn’t be possible to achieve without the accessibility to the data hub from Strava. (learn more on how you can extract your workout log from strava here)

Running Javascript to upload items to Dynamo DB on AWS cloud service. This would essentially help in scaling when we have bigger database. Learn more about the process on GitHub.

Design

The design elements are inspired by the actual Fitness Tracker applications like Apple Fitness and Strava. Using bold colors like mandarin orange contrasting with a divergent color like Persian blue. Typeface is the IBM monopoles which fits the funky-energetic mood and tone of this project.

Usage

I wanted to see multiple routes and see the pattern in each run all at once. However, this function wasn’t available on the app. This initial idea inspired me to design the overview map that I would be able to view my running map in Central Park using various colors! When clicking on each line, I can also see the performance at each point I run.
